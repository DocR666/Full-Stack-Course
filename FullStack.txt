Selectors ->
Select specific elements

Types->
Element selector: Selects all elements of specific type, e.g. h1, p, div, etc.

Class selector: Label multiple elements to style them consistently. Use a period (.)

ID selector: Style element that appears only once (ID is unique, class is not). Use #

Event handlers ->

onclick

const buttons = document.querySelectorAll(".alert-button");

buttons.forEach(button => {
    button.addEventListener("click", () => {
      alert("Button clicked!");
    });
  });

paragraph.addEventListener("mouseover", () => {
            paragraph.textContent = "You hovered over me!";
});

paragraph.addEventListener("mouseout", () => {
            paragraph.textContent = "Hover over me!";
});

Objects in JavaScript ->

let person = {
name: "Alice",
age: 30,
email: "alice@jmail.com"
};

To retrieve name - person.name/person["name"]

To remove a property - delete person.email;

Object.keys(person) - Output: ["name", "age", "country"]
Object.values(person) - Output: ["Alice", 31, "USA"]
Object.entries(person) - Output: [["name", "Alice"], ["age", 31], ["country", "USA"]]
